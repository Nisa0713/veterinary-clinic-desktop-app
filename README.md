# Veterinary Clinic Desktop Application

A desktop management system developed for veterinary clinics using C# and SQL.

## Features

- Patient management
- Pet owner records
- Doctor management
- Medication tracking
- Appointment scheduling
- Payment and invoice management

## Technologies

- C#
- .NET Windows Forms
- SQL Server

## Database

The project includes a SQL script (`VetClinicDB_Setup.sql`) to create the database structure.
